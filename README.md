# Learnix

Learnix is a free, open platform built to help diploma students with study materials, PYQs, entrance exam prep, and other academic resources.

The goal is simple: put everything a diploma student needs in one clean and easy-to-use website.

---

## What Learnix offers

- Subject-wise notes for diploma courses  
- Previous year question papers (PYQs)  
- Entrance exam preparation resources  
- E-books and reference material  
- Virtual lab access  
- Clean UI with light and dark mode  

---

## Tech Stack

- HTML  
- Tailwind CSS  
- JavaScript  
- Font Awesome  
- Google Fonts  

This project is completely frontend-based.

---

## How it works

- Static website built using Tailwind CSS  
- No backend or database  
- Contact and access forms handled using third-party form services  
- Dark mode preference saved using localStorage  

---

## Purpose

Learnix was created to help diploma students who struggle to find organized and reliable study resources online.  
All content is focused on practical learning and exam preparation.
